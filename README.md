# Simple Google OAuth Library


### Introduction

A light-weight & easy to use Google OAuth 2.0 library.


### Requirements

 - PHP >=5.3
 - PHP cURL extension
 - PHP JSON

### Installation
Just download the package & `include` it.

    <?php
        require 'src/SimpleGoogleOAuth.php';
    ?>

###Usage

See `example.php` for Usage.
